Host Compatibility Notes:
Pro Tools supports the following MIDI events from plug-ins:
  • NoteOn
  • NoteOff
  • Pitch bend
  • Polyphonic key pressure
  • Bank select (controller #0)
  • Program change (no bank)
  • Channel pressure
