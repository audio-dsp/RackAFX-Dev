Add a component to an instance of a component descriptor. 

Unlike with \hyperlink{a00087_a5ff114b8c4da2081515186f2faf65c8c}{A\+A\+X\+\_\+\+I\+Collection\+::\+Add\+Effect()}, the \hyperlink{a00096}{A\+A\+X\+\_\+\+I\+Effect\+Descriptor} does not take ownership of the \hyperlink{a00088}{A\+A\+X\+\_\+\+I\+Component\+Descriptor} that is passed to it in this method. The host copies out the contents of this descriptor, and thus the plug-\/in may re-\/use the same descriptor object when creating additional similar components.
